# AgmDirectionTest

Testing for agm-direction.

This project was generated by [Angular CLI](https://cli.angular.io/). 

## Installation

```bash
git clone https://github.com/explooosion/Agm-Direction.git
```

```bash
cd playground && yarn
```

## Build

**Root** directory:

```bash
yarn build
```

## Run

```bash
cd playground
```

```bash
yarn start
```
